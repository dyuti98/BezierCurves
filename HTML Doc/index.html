Introduction
This HTML page has been created for the Computer Graphics Assignment 3 Sem 1 2018-19 under Prof. Tathagata Ray.
We have implemented de Casteljau's algorithm for Bezier Curve Drawing using OpenGL.
Control points can be added, moved and deleted.
Right click on a point to go into edit mode. While in edit mode, one can delete the highlited point by pressing the delete key, or move the control point by clicking elsewhere.
THE PROGRAM
The assignment has three major folders: Assignment 3, Doxygen and HTML Doc.
The Assignment 3 directory contains the main.cpp, MidpointCircle.cpp and shaders.cpp.
All functions relevant to de Casteljau's algorithm are in main.cpp. We are drawing 2000 points (i.e. drawing a point for 2000 parametric values).
